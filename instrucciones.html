Instrucciones de Uso

🚀 Abrir Aplicación

📱 ¿Qué puedes hacer con esta aplicación?

💰 Aportes Familiares

Controla quien ha aportado dinero y cuánto falta por recibir.

💸 Gastos

Registra todos los gastos organizados por categorías.

📝 Tareas

Organiza las tareas pendientes con responsables y fechas.

👥 Invitados

Gestiona la lista de invitados y sus confirmaciones.

📊 Resumen

Ve el estado general de la organización de tu fiesta.

📱 Responsive

Funciona perfectamente en móviles y tablets.

🚀 Cómo Empezar

  1. Abre la aplicación: Haz clic en "Abrir Aplicación" arriba
  2. Agrega familiares: Ve a la pestaña "Aportes" y empieza a registrar
  3. Registra gastos: Anota todos los gastos estimados en "Gastos"
  4. Organiza tareas: Crea y asigna tareas en "Tareas"
  5. Gestiona invitados: Lleva el control de quién viene en "Invitados"
  6. Revisa progreso: Consulta el "Resumen" para ver el estado general

💾 Gestión de Datos

Tus datos están seguros: Todo se guarda localmente en tu navegador. Nadie más tiene acceso.

📊 Exportar a Excel

Descarga un archivo Excel con todos tus datos organizados en pestañas.

📈 Importar desde Excel

¿Ya tienes datos en Excel? Puedes importarlos directamente. El archivo debe tener estas hojas:

  • "Aportes de la Familia" - con datos de familiares
  • "Gastos de la Fiesta" - con gastos por categoría
  • "Tareas" - con tareas y responsables
  • "Invitados" - con lista de invitados

💾 Respaldos

Importante: Haz respaldos regulares usando "Exportar Base de Datos".

🌐 Para usar en GitHub Pages

Si quieres que múltiples familiares accedan a la aplicación:

  1. Sube estos archivos a un repositorio de GitHub
  2. Activa GitHub Pages en la configuración
  3. Comparte la URL con tu familia
  4. Cada quien puede usar la aplicación desde su dispositivo
URL de ejemplo:
https://tuusuario.github.io/organizador-fiesta/

📱 Compatible con Móviles

La aplicación está optimizada para:

  • 📱 Teléfonos móviles (Android/iOS)
  • 📱 Tablets
  • 💻 Computadoras de escritorio
  • 💻 Laptops

❓ Preguntas Frecuentes

¿Necesito internet para usar la aplicación?

Solo para cargarla la primera vez. Después funciona sin internet.

¿Se pierden mis datos si cierro el navegador?

No, los datos se guardan automáticamente en tu navegador.

¿Puedo usar la aplicación en diferentes dispositivos?

Sí, pero tendrás que exportar los datos de un dispositivo e importarlos en otro.

¿Es segura mi información?

Completamente. Todo se almacena localmente en tu dispositivo, no se envía a ningún servidor.
